标准 API 对接

外部系统调用 POST /api/print-jobs 传入数据，返回打印预览链接，对接简单、集成成本低。

📋

单据 & 标签打印

支持采购单、出库单等带明细表格的单据，也支持标签批量分页打印，适配多种纸张与标签尺寸。

🏷️

条码 / 二维码

内置条码与二维码组件，适用于仓储标签、资产标识、物流面单等场景，开箱即用。

多种打印方式

浏览器预览打印、远程静默打印、先预览再远程打印，灵活适配办公室与仓库等不同场景。

📦

开箱即用部署

传统痛点 LetsPrint 方案
改打印格式要发版、排期开发 业务人员自行设计模板，改完即生效
每种单据都要写一套打印代码 导入 JSON 自动识别字段，拖拽排版
标签、单据格式不统一 统一模板管理，支持多种纸张规格
仓库电脑需要静默出纸 Agent 远程投递，指定打印机自动打印
系统对接成本高、周期长 标准 REST API，返回预览链接，对接简单

三种打印模式

按场景选择最合适的出纸方式

browser

浏览器预览打印

打开预览页，操作员确认后通过浏览器打印，适合办公室日常单据。

remote

远程静默打印

通过 Agent 直接提交到目标电脑的打印机，无需人工干预，适合仓库流水线。

preview-remote

先预览再远程打印

操作员在预览页确认无误后，一键发送到远程打印机，兼顾准确与效率。
